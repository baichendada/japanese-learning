# 五十音打字练习 PRD / Design Spec

## 1. Product Summary

做一个面向日语五十音初学者的网页练习工具。产品第一屏直接进入打字练习，参考 dazidazi.com 的沉浸式练习交互：顶部状态仪表盘、中心题目线、空格开始、Tab 重练、右上角工具入口、右侧关卡切换面板。

本产品不以“课程首页”作为入口，而是默认读取本地进度并进入当前关卡。学习资料、易混淆假名、错题练习、导入导出和设置作为辅助入口放在右上角。

## 2. Goals

- 帮助已经接触过五十音但记不牢、容易混淆的用户快速建立“假名 -> 罗马音 -> 发音”的反应。
- 兼容完全零基础用户，通过右上角“自学”入口提供轻量学习材料。
- 提供清爽、低干扰、类似打字网站的练习体验。
- 通过固定关卡题组和渐进式解锁，让学习路径可控。
- 无账号保存进度，并支持导出/导入备份。

## 3. Non-Goals

- MVP 不做账号系统和云同步。
- MVP 不做真实日语单词系统，先以假名和固定题组为主。
- MVP 不做复杂听力训练；发音用于辅助记忆，不作为主要答题方式。
- MVP 不复制键盘指法训练逻辑；五十音场景下不默认展示指法手势图。

## 4. Target Users

主要用户是看过五十音表但经常忘记、混淆读音和罗马音的初学者。

次要用户是完全没学过五十音的人，他们可以从右上角“自学”入口进入基础说明，再回到练习流。

## 5. Core Experience

### 5.1 First Screen

打开网页后直接进入练习场：

- 左上角显示产品名。
- 顶部中间显示实时状态仪表盘。
- 右上角显示工具入口：自学、易混淆、错题、导出、导入、设置、全屏。
- 中间显示当前关卡的假名序列。
- 未开始时提示“点击或按空格开始输入”。
- 下方显示日语提示区。
- 再下方显示当前关卡名和切换关卡按钮。

### 5.2 Practice Flow

1. 页面启动后读取本地进度。
2. 自动加载当前应练关卡。
3. 用户点击页面或按空格开始。
4. 用户根据屏幕上的假名输入对应罗马音。
5. 每次按键播放打字机音效。
6. 单个假名输入正确后播放该假名发音。
7. 多假名题目时，按键只播放打字机音效，整题完成后播放完整读音。
8. 输入错误时标红当前输入位置，累计错误次数。
9. 达到通关条件后展示结算，并自动解锁下一关。
10. 用户可按 Tab 重练当前关。

## 6. Main UI Components

### 6.1 Dashboard

顶部中间仪表盘显示：

- 状态：预备、运行中、通过、失败
- 用时
- 正确率
- 错误次数，例如 `0/4`
- 速度，例如 `42.0 假名/分钟`

准确率决定是否通关，速度只影响星级或评价。

### 6.2 Kana Line

中心题目线显示当前关卡的假名序列。未开始时可轻微模糊，开始后清晰显示。当前输入位置需要高亮。

题目内容来自固定关卡题组。系统可以在内部调整展示顺序，但界面不显示内部排序说明。

### 6.3 Kana Helper

底部提示区替代 dazidazi 的键盘指法图。默认展示当前假名的记忆提示：

- 当前假名大字卡，例如 `け`
- 罗马音拆解，例如 `k + e = ke`
- 当前行小表，例如 `か ka / き ki / く ku / け ke / こ ko`
- 特殊拼法提醒，例如 `し = shi`、`ち = chi`、`つ = tsu`
- 发音重听入口

提示区可在设置里隐藏。

### 6.4 Level Switcher

当前关卡标题旁有切换按钮。点击后从右侧打开关卡面板，参考 dazidazi 的关卡列表体验。

关卡面板显示：

- 当前训练方案，例如“平假名 - 基础”
- 每关编号和名称
- 每关固定题组
- 通关状态
- 正确率
- 星级
- 当前关卡标记
- 未解锁状态和解锁条件
- “进入训练”操作

用户可以直接进入已解锁关卡。进入其他关卡不会清空原本进度。

## 7. Content Scope

MVP 包含平假名和片假名，二者分开训练、分开闯关。

第一阶段关卡按五十音行推进：

- 平假名：あ行、か行、さ行、た行、な行、は行、ま行、や行、ら行、わ行
- 片假名：ア行、カ行、サ行、タ行、ナ行、ハ行、マ行、ヤ行、ラ行、ワ行
- 每学几行后插入混合复习关
- 错题和易混淆假名提高后续辅助练习中的出现频率

真实日语单词可作为后续增强，不进入 MVP 主范围。

## 8. Level Rules

每关有固定题组，例如：

- `1/92 あ行：あ い う え お`
- `2/92 あ行复习`
- `3/92 か行：か き く け こ`
- `4/92 あ行 + か行复习`

通关条件：

- 一关固定题量，例如 20 题或等价长度
- 正确率达到 90% 才通关
- 错误次数超过限制则失败
- 速度不影响是否通关，只影响星级

星级规则：

- 1 星：达到通关准确率
- 2 星：达到通关准确率且速度达到基础目标
- 3 星：达到通关准确率且速度达到优秀目标

## 9. Audio Rules

- 每次按键播放打字机音效。
- 单假名关卡中，答对一个假名后播放该假名发音。
- 多假名题目中，每次按键只播放打字机音效，整题打对后播放完整读音。
- 当前题目提供重听入口。
- 设置中提供两个独立开关：按键音效、假名发音。

## 10. Local Progress

不登录，使用浏览器本地存储保存：

- 当前训练方案
- 当前关卡
- 已解锁关卡
- 每关最好成绩
- 每关星级
- 错题统计
- 易混淆假名统计
- 音效和提示设置

## 11. Import / Export

支持导出 JSON 进度文件，用于备份或换设备。

导入时先展示摘要，再让用户选择：

- 覆盖本地进度
- 合并进度

合并策略：

- 关卡解锁取更高进度
- 每关成绩保留更好成绩
- 错题统计累加或取更高权重
- 设置项以导入文件为准前需用户确认

## 12. Auxiliary Modules

### 12.1 自学

提供五十音基础表、平假名/片假名切换、发音播放和简单记忆说明。它是辅助入口，不替代主练习流。

### 12.2 易混淆

展示常见混淆组，例如：

- し / ち / つ 的特殊罗马音
- ソ / ン
- シ / ツ
- さ / ち 等形近或音近组合

用户可以进入专项练习。

### 12.3 错题

基于本地统计自动收集用户输错的假名，支持专项复练。错题练习不阻塞主线关卡。

## 13. Error Handling

- 用户输入错误时，当前输入位置标红并播放错误音。
- 错误次数超过限制时本关失败，显示重练入口。
- 发音资源加载失败时不阻塞练习，只隐藏或禁用重听入口。
- 导入文件格式错误时显示明确错误，不覆盖本地数据。
- 本地存储不可用时允许临时练习，但提示进度无法保存。

## 14. Testing Criteria

MVP 验收时至少覆盖：

- 首次打开能进入默认第一关。
- 有本地进度时能自动恢复当前关。
- 空格开始、Tab 重练可用。
- 正确输入、错误输入、通关、失败状态可用。
- 关卡切换面板可打开，已解锁关卡可进入，未解锁关卡不可进入。
- 平假名和片假名训练方案可切换。
- 错题统计能记录错误假名。
- 进度可导出，导入前有摘要，覆盖和合并都可用。
- 音效和发音开关生效。

## 15. Approved Direction

已确认的产品方向：

- 主体验参考 dazidazi 的沉浸式打字练习界面。
- 不做卡片式首页，打开即练。
- 日语提示区替代键盘指法图。
- 右上角放自学、易混淆、错题、导入导出和设置。
- 当前关卡旁提供切换按钮，打开右侧关卡列表。
- 关卡题组固定，用户界面不展示内部排序说明。
